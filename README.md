## ğŸš€ Getting Started

This template is designed to be incredibly easy to get up and running.

### Prerequisites

You only need a web browser to view this template. No special software or server setup is required for basic viewing.

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/Pedram-Mirkarimi/Website-Template.git
    ```
2.  **Navigate into the project directory:**
    ```sh
    cd Website-Template
    ```

### Running the Template

Simply open the `index.html` file in your preferred web browser.

```sh
# Example for opening on macOS/Linux
open index.html

# Example for opening on Windows
start index.html
```

Alternatively, you can use a live server extension in your code editor (like VS Code's Live Server) for automatic page refreshes during development.
